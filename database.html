Database development Unit 10: Database development

In this unit we had to learn about databases. Our scenario was that we work for a local doctor’s surgery and have been asked to develop a simple information management system.

For assignment A, we had to prepare a demonstration to show how databases are currently used in two different ways by organisations.

For assignment B, we had to design a database for doctor's surgery.

For assignment C, we had to develop the database. It had to be populated with a given data set. We also had to test it and fix all of the errors. Additionally, we had to ask for feedback to improve the database.

For assignment D, we had to review the finished relational database, evaluate it, justify and suggest improvements.
